Smurf Me UK!

Smurf.me.uk is a list of resources to help new and existing Ingress Resistance players get started with Ingress, or to get more involved in their local actions. All information on this site is "public" information that is not considered a security risk to the Ingress Resistance Community, however, as you get more involved you will find there are other resources and information that is considered Secure Information that is only shared with Trusted Ingress Resistance Agents and must not be shared with The Enlightened

Primarily, this site points at existing sources of information, however, there are a number of items which are original Smurf.me.uk content. These are considered to be released into the public domain, so you can do with them what you wish.

Getting Started

If you have been sent this link, it is most likely that the person who gave it you intended you to look at the following "Getting Started" resources:

Smurfling.Guide
The Smurfling Guide is a set of infographics designed to clearly explain the basics of Ingress to new players. It's a really great resource to get your head rond the game. It's been updated for Ingress Prime, the latest client software used to play the game.
FevGames Ingress Guide
FevGames work in partnership with Niantic (the developers of Ingress) to help run events, they have a more detailed Ingress guide which is useful when you want to dig in in more details
Evolve
The Project Evolve guide is designed to help Pokemon Go Players convert to the Ingress game and community. It covers key issues that differ between the play style required in PoGo vs Ingress
Communicate
The Communicate Guide is a primer on the different third party tools used to communicate in the Ingress Resistance Community

Top Tips

Try following these small pieces of advice to get going fast!

Glphy Hack Everything
At least hack every portal. But Glphy Hacking (long press to enable) gets you more Action Points and more gear.
Learn the Glyphs
Each glyph means something. As the portals get higher level, more Glyphs appear as you hack it. These form phrases. If you know the meaning of each Glphy it's easier to remember and get the sequence right. So if you know that the Glphys spelled out "Attack Enlightened Defend Resistance" it'll be easier to get them right and get more kit and AP from the portal.
Max AP
Every action earns you AP, each Resonator you add to a portal is a bit. But you get bonus AP for the first and last resonator, plus liking and fielding is more. So if you're in an area you can do it, make sure you full deploy, link and field as many portals as you can to level fast.
Get More Keys
Once two portals are fully deployed, you can link from one to the other. When standing at the first portal, you must have a key for the second. If you hack or glyph hack a portal there is a random chance it will drop one key. You can get more by drawing the "more" Glyph on the Glyph hack screen before it starts to show you the Glyph sequence for that hack. More keys, more links, more fields, more AP, more levels...
Connect Locally
Get involved in your local community, there are chat groups that will provide you with loads of help and advice!

Official Resources

The following is a list of official Niantic resources that are part of the Ingress platform you should be aware of.

Ingress Support
The official Ingress support site has many help articles and a mechanism for raising support tickets and issues with Niantic.
Intel Map
The Ingress Intel Map provides the ability to look beyond what you can see in your scanner and see the bigger picture. It allows you to look and see anywhere in the world, what portals are there, if they're deployed, linked, modified etc. It's all there.

Unofficial Resources

The tools provided by Niantic are somewhat limited, there are many more tools made by the community to help improve your Ingress experience. You should be very careful using third party tools, as some break the rules of Ingress and are considered cheating. The following tools are generally accepted as safe.

Agent Stats
This uses screenshots of your agent profile in Ingress to track your progress against in-game and expanded targets. Great when you're chasing your next levels!
IITC
IITC provides expansion of the Ingress Intel map to unlock it's potential
Glyphtionary
A Glyph Dictionary to help you learn the glyphs!

Join Us

If you want to get in touch with the active, coordinated Resistance in your area, there are a number of ways to get added to local chats. These chats will actively screen to make sure you are a genuine blue agent. You can use join.blue worldwide to find your local group. UK players can jump straight to ingresswith.us/local which will attempt to auto-detect your location and add you to the appopriate regional boot camp to be vetted.
